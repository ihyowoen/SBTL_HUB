# Future Card Standard — Full Schema Locked

이 문서는 **앞으로 생성될 카드**에 대한 고정 기준이다.

## 1. 신규 카드 표준

앞으로의 신규 카드는 **full schema only** 로 본다.

```json
{
  "id": "YYYY-MM-DD_REGION_NN",
  "region": "KR|US|CN|JP|EU|GL",
  "date": "YYYY-MM-DD",
  "cat": "...",
  "signal": "top|high|mid",
  "title": "...",
  "sub": "...",
  "gate": "...",
  "fact": "...",
  "implication": ["...", "..."],
  "urls": ["..."],
  "related": ["...", "..."]
}
```

- `id`: ID 형식 규정은 `docs/CARD_ID_STANDARD.md` 참조. 뉴스 날짜 기반의 `YYYY-MM-DD_REGION_NN` 포맷이 표준이다.
- `related`: 다른 카드와의 관계는 본문(fact / implication) 안에 ID를 하드코딩하지 않고, `related` 배열로 별도 관리한다. 프론트엔드에서 카드 하단 "관련 카드" 네비게이터로 렌더링한다. 양방향성 보완은 후속 스크립트로 처리한다.

---

## 2. Region Label Rule — Locked

region은 기사 출처 국가가 아니라 **사건의 직접 무대**로 붙인다.

- KR / US / CN / JP는 해당 국가가 직접 무대일 때만 사용
- 유럽 개별 국가 사건과 EU 제도 뉴스는 EU
- 두 개 이상 국가가 동등하게 핵심이면 GL
- 현재 배지 체계 밖 국가는 GL
- 애매하면 GL

### 2.1 엣지케이스 판례 — 반복 발생 패턴

이 판례는 룰이 추상적일 때 판단 시간을 절약하기 위해 실무 판정을 박제한 것이다.

**A. 해외 상장 · 해외 자본 조달**
- 본토 기업의 HK · SG · US 등 해외 상장/증자는 **기업 본국 기준**.
- 예) 중국 본토 기업의 HK IPO → `CN` (사업 무대가 실질 무대)
- 예) 한국 기업의 NASDAQ / NYSE 상장 → `KR`
- 예) 유럽 기업의 해외 상장 → `EU`
- 근거: 자본 조달 주체의 사업 활동지가 실질 무대이며, 거래소는 도관.

**B. 정책 검토 · 정책 보도 단계**
- 국가 X의 정책 검토 · 초기 보도는 실행 전이라도 **발화지 기준**.
- 예) 중국의 대미 태양광 설비 수출제한 "검토" → `CN`

**C. 합작 · JV**
- 두 국가 지분 · 경영 동등 → `GL`
- 한 쪽이 지배적(>50% 지분 또는 운영 주도) → 지배 쪽 국가

**D. 다국적 제품 · 플랫폼 런칭**
- 최초 런칭 시장 기준 (롤아웃 일정 아니라 최초 시장)

**E. 출처 매체와 사건 무대가 다른 경우 (원 룰 핵심 타겟)**
- 한국 매체가 중국 뉴스 보도 → `CN`
- 일본 매체가 유럽 뉴스 보도 → `EU`
- 즉, 매체 국적은 region 판정에 무관.
